# Frontend Rules

- Use Tailwind CSS only — never inline styles, never .css files
- Use functional React components only — no class components
- Use Fetch API only — never import or use Axios
- All API calls must live in /src/api/ — never directly in components
- All business logic must live in /src/hooks/ — never in pages or components
- Keep components under 100 lines — split if larger
- UI labels and text must be in French
- Variable names, function names, file names must be in English
- Use Lucide React for all icons
- Use Headless UI for all modals and dropdowns
