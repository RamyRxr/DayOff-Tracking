# Frontend Design Skill

Auto-invoked when building any UI component or page.

## Design Principles
- Clean, professional, minimal — this is an internal enterprise tool
- Primary color: deep navy blue (#1e3a5f) — professional, trustworthy
- Accent color: amber (#f59e0b) — for warnings, block buttons
- Success color: emerald (#10b981) — for active status, confirms
- Danger color: red (#ef4444) — for blocked status, destructive actions
- Background: light gray (#f8fafc) — easy on the eyes for long sessions
- Font: system font stack — fast loading, familiar

## Component Patterns
- Cards have subtle shadow and rounded corners (rounded-lg)
- Modals are centered with dark overlay backdrop
- Buttons always show their state: disabled = gray + cursor-not-allowed
- Block button must be visually distinct — red, bold
- Tables have alternating row colors for readability
- Status badges: colored pill shape (rounded-full)
